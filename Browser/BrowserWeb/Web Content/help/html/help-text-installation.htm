Installing the T24 Online HelpText

The T24 Online Help Text is delivered with the T24 release as a Zip file e.g. r06HelpText.zip.

Within this Zip file the following files are also included:

  • portal-help-index.xml
  • portal-help-menu.xml

The table below shows the location of where these files should be installed.

Release Documentation

Web Server Location

Portal HelpText ZIP file containing the XML HelpText files for each T24 application.

Any location accessible by the web server.

The location of the unzipped HelpText file should be specified in the T24 Browser file “helpParameters.xml”.  Please refer to the T24 Browser User Guide for information on how to configure this file.

Portal Index XML file – “portal-help-index.xml”.

Contains an index of the HelpText files.

On the web server in the “help” directory under the BrowserWeb web application installation.

(ensure that the original “portal-help-index.xml” is overwritten with the version at accompanied the Help Text).

Portal Menu XML file – “portal-help-menu.xml”.

Contains the menu of HelpText files.

On the web server in the “help” directory under the BrowserWeb web application installation.

(ensure that the original “portal-help-menu.xml” is overwritten with the version at accompanied the Help Text).

Configuring ‘helpParameters.xml’

The ‘helpParameters.xml’ file found in the created T24 Browser directory requires configuring. 

This XML file contains a number of parameters used to configure the HelpText system. 

The settings are as follows:

XML Tag

Description

Example Values

viewLocation

Specifies a directory where the HelpText files can be found for viewing.

Multiple ‘viewLocation’ nodes are supported.

D:\T24\HelpText\

editLocation

Specifies a directory where the HelpText files can be found for editing.

Multiple ‘editLocation’ nodes are supported.

D:\T24\HelpText\

language

Specifies the languages supported for HelpText files.

Multiple ‘language’ nodes are supported.

GB

FR

imagesDir

Specifies the base directory that HelpText field images are stored.

The directory must be accessible by the web server.

This can be a relative path or a full path.

help/images

C:\help\images

editable

Specifies whether the HelpText Files can be edited.

YES

NO

devMode

Specifies whether the HelpText system is currently being used in a Development mode.  Live sites should always have this set to NO. 

YES

NO

 


 

 

Once these files have been installed, restart the web server to ensure the files are accessible via a browser session. 

The T24 Portal documentation can be accessed using the following URL:

http://localhost:8080/BrowserWeb/portal/portal.htm

 (where "localhost" is the name or IP address of the web server, and "8080" is the port number of the web server).

The "T24 Portal" can also be accessed from the "Tools" menu of T24 Browser and the "T24 Media Center" of the T24 ToolBox product.
