Browser Tour - Welcome


Browser is the  browser based front end for Temenos T24.

Every T24 application, version, and enquiry will be launched in a separate window so that you can arrange the screen as you wish, with Browser taking up as much or as little space as you need.

As you might expect, things are a little different in a browser and this tour is designed to show where your favourite features have moved to, to highlight the differences between Browser and Desktop, and to introduce some of the new features. Click the navigation bar above to start the tour.
