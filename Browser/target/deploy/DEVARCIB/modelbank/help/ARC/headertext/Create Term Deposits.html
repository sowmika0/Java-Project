Create Term Deposits

List of your various accounts along with account description is displayed here.You can click on the account hyperlink to create a new term deposit with the Bank. The amount of deposit & deposit duration can be selected in the next screen upon submitting which a confirmation screen of the deposit agreement is displayed. Upon selecting the 'Confirm' option from the confirmation screen of the new term deposit agreement, the new term deposit should be created with the Bank and displayed with a reference no. in the transaction preview screen.
