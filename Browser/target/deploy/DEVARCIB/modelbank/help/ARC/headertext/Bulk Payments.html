Bulk Payments
Validate Bulk Payments :The Bulk Master & Bulk Item records that needs to be validated by the Clerk is available in this page. Options to create new Bulk Items via the Beneficiary is available. Also it is possible to Amend, see & add new bulk items from this page.

Amend Authorised Payments :This page displays the list of authorised Bulk Payment records which may be modified by the Clerk until the processing date is reached.

View Processed Bulk Records :The Bulk Payment records that have been processed upon reaching the processing date is displayed in this page.
