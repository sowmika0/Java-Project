Installing the T24 Online Documentation

The documentation for T24 Release is delivered as a Zip file e.g. T24r06doc.zip.

This file is supplied along with the T24 Release.

To Install the T24 Online Documentation unzip this file within the following location:

..\BrowserWeb\doc\

 (where BrowserWeb is the location of the T24 Browser Web Application)

During installation you will be prompted to overwrite/replace the 'portal-doc-menu.xml' file.
Ensure that this is accepted.

The Zip file contains documents in PDF format such as:

  • Release Notes (including installation instructions)
  • Defects List
  • Highlights
  • User Guides

These documents are viewable via the "T24 Portal" which is accessible via the T24 Browser product. To view these documents prior to T24 Browser installation please unzip the file to a convenient location. Navigate to the required user guide and open using a PDF viewer.

Once T24 Browser is installed, please refer to the section "Installing T24 HelpText and Online Documentation" within the "Browser Installation & Configuration" document for further details.

Once installed, and the T24 Browser web application is running, the "T24 Portal" can be accessed using the following URL :-

http://localhost:8080/BrowserWeb/portal/portal.htm

 (where "localhost" is the name or IP address of the web server, and "8080" is the port number of the web server).

The "T24 Portal" can also be accessed from the "Tools" menu of T24 Browser and the "T24 Media Center" of the T24 ToolBox product.
